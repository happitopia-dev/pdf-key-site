Password Lock for PDFs

送る前のPDFに、すぐ鍵をかける。

PDF Keyは、PDFを選び、パスワードを入力し、保存または共有するためのiPhoneアプリです。余計な編集機能を挟まず、必要な保護だけをすばやく終えられます。

App Storeで見る サポートを見る プライバシーポリシー

迷わない3ステップ

PDFを選択し、新しいパスワードを入力し、パスワード付きPDFとして保存または共有します。

ファイルは手元に

PDFやパスワードをHappitopiaのサーバーへアップロードしません。

送付前のひと手間に

見積書、控え、配布資料など、共有前に「開くパスワード」を付けたい場面に向いています。

Install

PDFを守る作業を、いつもの共有フローに。

iPhoneのファイル選択や共有から使いやすい、軽いPDF保護アプリです。

PDF Key is a simple iPhone app for protecting PDF files with an open password. Choose a PDF, enter a password, then save or share the protected file.
